This repository is Go-first with local Node packages under `runtime/` and `e2e/`.

## Prerequisites

- Go 1.24+
- Node.js 18+ (this repo currently runs with Node 25)
- `pnpm` 10+

## Build the binary

```bash
go build -o tsgonest ./cmd/tsgonest
```

## Run against fixtures

```bash
./tsgonest --project testdata/nestjs/tsconfig.json --config testdata/nestjs/tsgonest.config.json
```

The command emits JS via `tsgo`, then generates companion files, manifest, and OpenAPI.

## Common test commands

```bash
go test ./internal/...
```

```bash
cd e2e && pnpm run test --run
```

:::tip
`just` is intentionally not required for this repo. Use raw `go` and `pnpm` commands.
:::
